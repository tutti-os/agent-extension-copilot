# GitHub Copilot CLI package

The host-derived Agent Target is `extension:copilot`, the runtime provider is
`acp:copilot`, and the signing key ID is `tutti-copilot-release-v1`. Manifest v2
stores the stable `copilot` agent key; it does not accept provider or Target
fields. Keep all package content declarative and do not add commands, tools,
models, or permissions without official source or sanitized ACP probe evidence.

`icon.svg` is the colored shared branded identity for the Provider Rail,
conversation headers, Message Center, and mentions. `mask-icon.svg` is the
transparent conversation-row mask glyph.
